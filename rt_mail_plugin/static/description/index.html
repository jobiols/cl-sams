Overview

A Tool that Allow users to receive and send emails from a variety of email platforms.

In Odoo, the email management system provides a very basic and straightforward tool for managing emails. The user may also manage their inbox, sent things and drafts.

Features

Send/Receive Emails

Users will easily be able to send and receive emails using this plugin.

Reply/Forward Emails

Users will be able to reply to and forward emails just like it is done in Gmail.

Maintain Inbox

Users will be able to maintain an inbox within Odoo. Managing all emails in one place would now be made easier where followup of email activities will also be easier.

Maintain Send Items

Users will be able to maintain, manage and see all sent items in one place. Follow up activities of all sent emails will also be easier to manage.

Categories Based On Labels/Tags

Users will be able categorize all emails by applying tags and labels to every email. This makes separating emails and finding the ones needed easier to search for.

Activity Logs

Users will be able to see all the activities related to a particular email in the pan below the email, where all required information will be available there.

Screenshots

Compose Email

To compose email, you'll be using the compose window. This is where you'll add the email address of the recipient(s), the subject, and the body of the email, which is the message itself. You'll also be able to add various types of text formatting, as well as one or more attachments

Email Records

Email records list view will contain all records with recipient(s), sender, subject, date, type and the optionals attributes (priority and tags). User can have a quick look on all the email records.
